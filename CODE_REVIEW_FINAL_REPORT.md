## 发现的主要问题

### 1. 工具参数解析错误
**问题描述**: 查询"12加8等于多少"匹配了错误的模式，导致参数为空
**根本原因**: 意图识别器的模式顺序不当，通用模式`(.*)等于多少`在具体模式`(\d+)(加上|加|\+)(\d+)`之前匹配
**影响**: 工具执行失败，错误信息：`calculator() missing 1 required positional argument: 'expression'`

### 2. 工具结果集成失败
**问题描述**: API响应没有正确使用工具执行结果
**根本原因**: 
1. 上下文压缩方法可能移除工具结果标记
2. 系统消息不够明确，模型没有正确遵循指令
**影响**: 工具执行成功但API响应没有使用工具结果

### 3. 上下文压缩问题
**问题描述**: 压缩方法可能截断工具结果部分
**根本原因**: 简单的开头+结尾压缩策略可能移除中间的工具结果
**影响**: 工具结果标记被移除，API无法检测到工具结果

### 4. 系统消息不明确
**问题描述**: 模型没有正确使用工具结果中的数字
**根本原因**: 系统消息指令不够明确，模型倾向于生成通用的星露谷导游回复
**影响**: 工具结果没有被正确使用

**文件**: `src/tools/enhanced_tool_manager.py`

```python
    r"(\d+)(减去|减|\-)(\d+)",

    r"(\d+)(除以|除|/)(\d+)",

    r"(计算|算一下|计算一下)(.*)",

```
