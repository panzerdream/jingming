# 星露谷助手 RAG 系统启动指南

## 项目结构
```
agent-rag/
├── backend/          # FastAPI 后端服务
├── frontend/         # Vue 3 前端界面
├── src/              # 核心 RAG 逻辑
├── data/             # 知识库数据
└── requirements.txt  # Python 依赖
```

## 环境要求
- Python 3.9+
- Node.js 16+

## 一、后端启动

### 1. 安装 Python 依赖
```bash
cd agent-rag
pip install -r requirements.txt
cd backend
pip install -r requirements.txt
```

### 2. 配置阿里云百炼 API
确保已配置好阿里云百炼 API 密钥（在 src/api/bailian_api.py 中）

### 3. 启动后端服务
```bash
cd agent-rag/backend
python main.py
```
后端服务将在 `http://localhost:8001` 启动

## 二、前端启动

### 1. 安装 Node.js 依赖
```bash
cd agent-rag/frontend
npm install
```

### 2. 启动前端开发服务器
```bash
npm run dev
```
前端界面将在 `http://localhost:5173` 或可用端口启动

## 三、完整启动流程

### 方式一：分别启动（推荐用于开发调试）

**终端 1 - 启动后端：**
```bash
cd agent-rag/backend
python main.py
```

**终端 2 - 启动前端：**
```bash
cd agent-rag/frontend
npm run dev
```

### 方式二：构建生产版本

**构建前端：**
```bash
cd agent-rag/frontend
npm run build
```

## 注意事项

1. **确保后端先启动**：前端需要连接到后端 API
2. **向量数据库**：首次使用需要先运行 `src/main.py` 创建向量数据库
3. **端口占用**：如果端口被占用，后端默认使用 8001，前端默认使用 5173
4. **OpenMP 问题**：如果遇到 OpenMP 错误，请设置环境变量：
   ```bash
   export KMP_DUPLICATE_LIB_OK=TRUE
   ```

## 知识库文件

知识库文件位于 `data/knowledge/` 目录下：
- game.md - 游戏相关知识
- location.md - 地点信息
- npc.md - NPC 信息

可以随时添加或修改这些文件，然后重新运行 `src/main.py` 更新向量数据库。

## 前端界面调整

前端界面文件位于 `frontend/src/App.vue`，可以根据需要调整：
- 字体样式
- 颜色主题
- 布局设置

修改后无需重启，Vite 会自动热更新。
